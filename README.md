# 🔐 Diffie-Hellman Demo — Projet RP2026

## Description
Démonstration pédagogique interactive du protocole **Diffie-Hellman** en langage **Rust**.  
Projet réalisé dans le cadre du cours **Réseaux et Protocoles — 2026**  
à destination des lycéens du **Lycée Sainte-Marie de Cocody, Abidjan**.

## Auteur
**KOUADIO KOUASSI HIPOLITE**  
Université / UFRMI — Mai 2026

---

## 🎯 Objectifs pédagogiques
- Comprendre le fonctionnement du protocole Diffie-Hellman
- Visualiser l'échange de clés entre deux parties (Louis et Charles)
- Montrer pourquoi Ismaël (l'attaquant) ne peut pas retrouver le secret
- Répondre à la question de Bobo : **quelle est la frontière d'efficacité selon le choix des nombres premiers ?**

---

## 🚀 Lancer la démo

### Prérequis
- Rust installé (version 1.95.0 ou supérieure)
- Cargo (inclus avec Rust)

### Installation
```bash
git clone https://github.com/TON_USERNAME/diffie_hellman_demo.git
cd diffie_hellman_demo
cargo run
```
